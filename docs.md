## Turn on the Clock

1) Set the switch to the *on* or the *wake word* position. 
2) Plug in the power cable. The clock OS will boot up and the clock application will automatically start after a few minutes.

    > Note, if the clock is pluged in with the switch set to *off*, the clock will boot up and then shutdown immediately once it has booted.
